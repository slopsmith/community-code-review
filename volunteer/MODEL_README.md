# Community Code Review — Model Storage

This folder contains AI model files used by the **Community Code Review** volunteer container.

## What is this?

The files in this directory are large language model (LLM) weights downloaded by the Community Code Review volunteer Docker container. The container uses these models to perform automated code reviews on GitHub pull requests for open-source projects.

## Where did this come from?

Models are downloaded from [Hugging Face](https://huggingface.co/) or another source specified by the coordinator. The specific model and version depend on what the project coordinator has configured.

## Can I delete these files?

**Yes, but with caveats:**

- ✅ **Safe to delete when the container is stopped** — the model will be re-downloaded automatically the next time the container starts.
- ⚠️ **Re-downloading takes time and bandwidth** — a ~19GB model can take 10–30 minutes to download depending on your connection.
- ✅ **Safe to move to another drive** — just update the volume mount path in your Docker run command.

## How do I recover disk space temporarily?

1. Stop the volunteer container: `docker stop code-review-volunteer`
2. Delete the `.gguf` files in this folder

That's it. The model will re-download on next container start

## How do I remove everything permanently?

1. Stop and remove the container: `docker stop code-review-volunteer && docker rm code-review-volunteer`
2. Delete this entire folder
3. Remove the Docker image if desired (~6GB): `docker rmi ghcr.io/slopsmith/volunteer:latest`
